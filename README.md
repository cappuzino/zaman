# Zaman
**by Rosenbrae**

### A living, temporal memory layer for AI — memory that does not reset, but continues.

Most AI memory is a search index on a stateless model: it answers, then forgets.
**Zaman** is a time-aware memory that preserves the past instead of overwriting it, and lets
AI agents and their human partner share one evolving body of knowledge that **compounds** over
time instead of resetting.

**What makes it different**

- 🕓 **Time-aware truth** — every memory knows *when* it was believed; yesterday's truth is kept, not lost.
- 📌 **Nothing is overwritten** — append-only and fully auditable; the past stays true for its own time.
- 🔗 **Meaning in connections** — memories are linked into a web, not stored as isolated facts.
- 🧬 **Continuity across agents** — each new agent inherits context and lessons instead of starting blank.
