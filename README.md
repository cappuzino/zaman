# Zaman
**by Rosenbrae**

### A living, biomimetic, neuro-symbolic cognitive architecture with a temporal memory layer.

Yesterday's truth can change tomorrow. **Zaman** is time-aware memory that
preserves the past instead of overwriting it — built to change how decisions
are made, not merely to recall them.
